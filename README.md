## Development
### Installing Dependencies
From within the root directory:

```sh
npm install
```

### Running Application
start local database
```sh
mysql.server start
```
build bundle for client, and keep running in terminal
```sh
npm run compile
```
run server, and keep running in terminal
```sh
npm start
```
