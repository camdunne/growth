# Growth

## Getting Started
### Installation
```
npm install
```
### Requirements
> Node.js
> MySQL

### Running Application
###### start local database
````
mysql.server start
````
###### build bundle for client, and keep running in terminal
````
npm run compile
````
###### run server, and keep running in terminal
````
npm start
````
